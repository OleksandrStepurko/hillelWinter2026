# MapStruct Showcase (Maven)

Минимальный проект (консоль + тесты), демонстрирующий ключевые фичи MapStruct.

## Что показывает
- Маппинг вложенных объектов, списков и `Map<String, String>`
- Форматирование дат и денег
- `@AfterMapping` (подсчёт итога заказа, разбор `fullName`)
- Обновление существующих сущностей через `@MappingTarget`
- Общая конфигурация через `@MapperConfig`
- Пример контекста для обхода циклических ссылок

## Запуск
```bash
mvn -q -DskipTests=false test
```
